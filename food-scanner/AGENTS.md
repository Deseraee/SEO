# Expo SDK 54

This project runs on **Expo SDK 54** (downgraded from the scaffolded SDK 56 to match
the Expo Go app, which maxes out at SDK 54).

- Read the versioned docs at https://docs.expo.dev/versions/v54.0.0/ before writing Expo code.
- Do NOT upgrade the Expo SDK past 54 — Expo Go cannot run a newer SDK.
- See ../PROJECT.md for the full project spec, work split, and the data contract.
